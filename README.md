# python-package-github-release-test

📦 Python package GitHub Release test

## Prerequisites

- [Python](https://www.python.org/)
- [uv](https://docs.astral.sh/uv/) or pip

## Install

Install with uv:

```sh
uv add python-package
```

Or install with pip:

```sh
pip install python-package
```

## Usage

Print the greeting:

```py
from python_package import hello

print(hello())
```
